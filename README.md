# navyk
NAVYK — SaaS-платформа для аналитики и развития карьеры студентов. Соединяем университеты, студентов и работодателей в едином цифровом пространстве. Университеты получают аналитику, студенты — карьерные рекомендации, работодатели — доступ к талантам. Проект для рынка СНГ на Next.js, NestJS и PostgreSQL.
